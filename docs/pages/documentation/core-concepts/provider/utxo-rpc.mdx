## UTxORPC

The Mavis SDK UTxO RPC Provider offers a JavaScript/TypeScript interface for interacting with UTxO-based blockchains using Mavis SDK. By leveraging gRPC and UTxO RPC, it allows developers to seamlessly integrate Cardano blockchain interactions into their decentralized applications (dApps).

Learn more at: https://github.com/utxorpc/mavis-sdk-provider
