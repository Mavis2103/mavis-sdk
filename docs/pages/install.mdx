# Install Mavis SDK

Mavis SDK is a powerful, modular framework for building Cardano applications off-chain. It provides developers with type-safe interfaces and utilities to interact with the Cardano blockchain, create transactions, and manage wallets without dealing with low-level complexities.

## Prerequisites

Before installing Mavis SDK, ensure your environment meets the prerequisites:

- **Node.js**: Version 14 or higher
- **Package Manager**: npm, yarn, pnpm, or bun

## Installation

Choose your preferred package manager to install Mavis SDK:

<Tabs items={["pnpm", "npm", "yarn", "bun"]}>
  <Tabs.Tab>```bash pnpm i @mavis-sdk/lucid ```</Tabs.Tab>
  <Tabs.Tab>```bash npm i @mavis-sdk/lucid ```</Tabs.Tab>
  <Tabs.Tab>```bash yarn add @mavis-sdk/lucid ```</Tabs.Tab>
  <Tabs.Tab>```bash bun i @mavis-sdk/lucid ```</Tabs.Tab>
</Tabs>
